# PU Machine Operation Parameters Calculator

A specialized calculator for polyurethane spraying process operations.

## Features

- **Part 1**: Basic formula calculations (O = C × L)
- **Part 2**: Test data analysis
- **Part 3**: Parameter adjustment
- **Multi-language support**: English, Vietnamese, Thai, Chinese
- **Clipboard functionality**: Automatically copies results for easy sharing
- **Mobile-friendly**: Responsive design works on all devices

## How to Use

### Getting Started
1. Visit https://binhdnguyen.github.io/tinhtoan/ or open `index.html` in any web browser
2. Select your preferred language from the header (En/Vi/Th/Cn)
3. Choose the appropriate calculator part using the navigation buttons

### Part 1: Basic Calculator
- Enter values for 3 parameters and leave 1 as "0" to calculate the missing value
- **Ratio is mandatory** and cannot be blank or zero
- Uses the formula: O = C × L (Output = Consumption × Line Speed)
- Provides material component breakdown for different time intervals

### Part 2: Test Data Analysis  
- Input actual test data from your polyurethane spraying process
- Required fields: Line Speed, Test Time, Polyol Weight, Isocyanate Weight
- Optional: Polyol RPM and Iso RPM for more accurate calculations
- Calculates ratio, output, and consumption based on real data

### Part 3: Parameter Adjustment
- Uses results from Part 2 to calculate new parameters
- Input desired new line speed, consumption, and ratio
- Provides adjusted component weights and RPM settings
- Shows time-based breakdown for operational planning

### Results
- All calculations are automatically copied to your clipboard
- Results include detailed breakdowns and material analysis
- Switch languages anytime - all results will be in your selected language

## Tips for Best Results
- Always ensure the ratio field in Part 1 has a valid positive number
- Use Part 2 to establish baseline parameters from actual test data
- Part 3 is most accurate when used with Part 2 results
- Results are formatted for easy sharing with your team

## Technical Requirements
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - just open the HTML file
- Works offline after initial load

# Máy Tính Thông Số Hoạt Động Máy Phun PU

Máy tính chuyên dụng cho quá trình hoạt động phun polyurethane.

## Tính Năng

- **Phần 1**: Tính toán công thức cơ bản (O = C × L)
- **Phần 2**: Phân tích dữ liệu test
- **Phần 3**: Điều chỉnh thông số
- **Hỗ trợ đa ngôn ngữ**: Tiếng Anh, Tiếng Việt, Tiếng Thái, Tiếng Trung
- **Tính năng clipboard**: Tự động sao chép kết quả để chia sẻ dễ dàng
- **Thân thiện với mobile**: Thiết kế responsive hoạt động trên mọi thiết bị

## Cách Sử Dụng

### Bắt Đầu
1. Truy cập https://binhdnguyen.github.io/tinhtoan/ hoặc mở `index.html` trong bất kỳ trình duyệt web nào
2. Chọn ngôn ngữ ưa thích của bạn từ header (En/Vi/Th/Cn)
3. Chọn phần máy tính phù hợp bằng các nút điều hướng

### Phần 1: Máy Tính Cơ Bản
- Nhập giá trị cho 3 thông số và để 1 thông số là "0" để tính toán giá trị còn thiếu
- **Tỷ lệ là bắt buộc** và không thể để trống hoặc bằng không
- Sử dụng công thức: O = C × L (Sản lượng = Tiêu hao × Tốc độ dây chuyền)
- Cung cấp phân tích thành phần vật liệu cho các khoảng thời gian khác nhau

### Phần 2: Phân Tích Dữ Liệu Test
- Nhập dữ liệu test thực tế từ quá trình phun polyurethane của bạn
- Các trường bắt buộc: Tốc Độ Dây Chuyền, Thời Gian Test, Khối Lượng Polyol, Khối Lượng Isocyanate
- Tùy chọn: Polyol RPM và Iso RPM để tính toán chính xác hơn
- Tính toán tỷ lệ, sản lượng và tiêu hao dựa trên dữ liệu thực

### Phần 3: Điều Chỉnh Thông Số
- Sử dụng kết quả từ Phần 2 để tính toán các thông số mới
- Nhập tốc độ dây chuyền, tiêu hao và tỷ lệ mong muốn mới
- Cung cấp khối lượng thành phần đã điều chỉnh và cài đặt RPM
- Hiển thị phân tích theo thời gian để lập kế hoạch vận hành

### Kết Quả
- Tất cả các phép tính đều được tự động sao chép vào clipboard của bạn
- Kết quả bao gồm phân tích chi tiết và phân tích vật liệu
- Chuyển đổi ngôn ngữ bất cứ lúc nào - tất cả kết quả sẽ bằng ngôn ngữ bạn đã chọn

## Mẹo Để Có Kết Quả Tốt Nhất
- Luôn đảm bảo trường tỷ lệ trong Phần 1 có số dương hợp lệ
- Sử dụng Phần 2 để thiết lập các thông số cơ sở từ dữ liệu test thực tế
- Phần 3 chính xác nhất khi được sử dụng với kết quả Phần 2
- Kết quả được định dạng để dễ dàng chia sẻ với nhóm của bạn

## Yêu Cầu Kỹ Thuật
- Bất kỳ trình duyệt web hiện đại nào (Chrome, Firefox, Safari, Edge)
- Không cần cài đặt - chỉ cần mở file HTML
- Hoạt động offline sau khi tải ban đầu

# เครื่องคำนวณพารามิเตอร์การทำงานเครื่องพ่น PU

เครื่องคำนวณเฉพาะสำหรับการดำเนินงานกระบวนการพ่นโพลียูรีเทน

## คุณสมบัติ

- **ส่วนที่ 1**: การคำนวณสูตรพื้นฐาน (O = C × L)
- **ส่วนที่ 2**: การวิเคราะห์ข้อมูลทดสอบ
- **ส่วนที่ 3**: การปรับพารามิเตอร์
- **รองรับหลายภาษา**: อังกฤษ, เวียดนาม, ไทย, จีน
- **ฟังก์ชันคลิปบอร์ด**: คัดลอกผลลัพธ์โดยอัตโนมัติเพื่อการแชร์ที่ง่าย
- **เป็นมิตรกับมือถือ**: การออกแบบที่ตอบสนองทำงานบนทุกอุปกรณ์

## วิธีใช้งาน

### เริ่มต้น
1. เยี่ยมชม https://binhdnguyen.github.io/tinhtoan/ หรือเปิด `index.html` ในเว็บเบราว์เซอร์ใดก็ได้
2. เลือกภาษาที่คุณต้องการจากส่วนหัว (En/Vi/Th/Cn)
3. เลือกส่วนเครื่องคำนวณที่เหมาะสมโดยใช้ปุ่มนำทาง

### ส่วนที่ 1: เครื่องคำนวณพื้นฐาน
- ใส่ค่าสำหรับพารามิเตอร์ 3 ตัวและเหลือ 1 เป็น "0" เพื่อคำนวณค่าที่หายไป
- **อัตราส่วนเป็นข้อบังคับ** และไม่สามารถเว้นว่างหรือเป็นศูนย์ได้
- ใช้สูตร: O = C × L (ผลผลิต = การใช้ × ความเร็วสาย)
- ให้การวิเคราะห์องค์ประกอบวัสดุสำหรับช่วงเวลาต่างๆ

### ส่วนที่ 2: การวิเคราะห์ข้อมูลทดสอบ
- ป้อนข้อมูลทดสอบจริงจากกระบวนการพ่นโพลียูรีเทนของคุณ
- ฟิลด์ที่จำเป็น: ความเร็วสาย, เวลาทดสอบ, น้ำหนัก Polyol, น้ำหนัก Isocyanate
- ทางเลือก: Polyol RPM และ Iso RPM เพื่อการคำนวณที่แม่นยำยิ่งขึ้น
- คำนวณอัตราส่วน, ผลผลิต และการใช้จากข้อมูลจริง

### ส่วนที่ 3: การปรับพารามิเตอร์
- ใช้ผลลัพธ์จากส่วนที่ 2 เพื่อคำนวณพารามิเตอร์ใหม่
- ป้อนความเร็วสาย, การใช้ และอัตราส่วนใหม่ที่ต้องการ
- ให้น้ำหนักองค์ประกอบที่ปรับแล้วและการตั้งค่า RPM
- แสดงการแบ่งตามเวลาสำหรับการวางแผนการดำเนินงาน

### ผลลัพธ์
- การคำนวณทั้งหมดจะถูกคัดลอกไปยังคลิปบอร์ดของคุณโดยอัตโนมัติ
- ผลลัพธ์รวมถึงการแบ่งรายละเอียดและการวิเคราะห์วัสดุ
- เปลี่ยนภาษาได้ตลอดเวลา - ผลลัพธ์ทั้งหมดจะเป็นภาษาที่คุณเลือก

## เคล็ดลับสำหรับผลลัพธ์ที่ดีที่สุด
- ตรวจสอบให้แน่ใจว่าฟิลด์อัตราส่วนในส่วนที่ 1 มีตัวเลขบวกที่ถูกต้องเสมอ
- ใช้ส่วนที่ 2 เพื่อสร้างพารามิเตอร์พื้นฐานจากข้อมูルทดสอบจริง
- ส่วนที่ 3 แม่นยำที่สุดเมื่อใช้กับผลลัพธ์ส่วนที่ 2
- ผลลัพธ์ถูกจัดรูปแบบเพื่อแชร์กับทีมของคุณได้ง่าย

## ความต้องการทางเทคนิค
- เว็บเบราว์เซอร์สมัยใหม่ใดๆ (Chrome, Firefox, Safari, Edge)
- ไม่จำเป็นต้องติดตั้ง - เพียงเปิดไฟล์ HTML
- ทำงานออฟไลน์หลังจากโหลดครั้งแรก

# PU喷涂机操作参数计算器

专为聚氨酯喷涂工艺操作而设计的计算器。

## 功能特点

- **第1部分**：基础公式计算 (O = C × L)
- **第2部分**：测试数据分析
- **第3部分**：参数调整
- **多语言支持**：英语、越南语、泰语、中文
- **剪贴板功能**：自动复制结果便于分享
- **移动端友好**：响应式设计适用于所有设备

## 使用方法

### 开始使用
1. 访问 https://binhdnguyen.github.io/tinhtoan/ 或在任何网络浏览器中打开 `index.html`
2. 从页头选择您首选的语言 (En/Vi/Th/Cn)
3. 使用导航按钮选择合适的计算器部分

### 第1部分：基础计算器
- 输入3个参数的值，将1个参数留为"0"以计算缺失值
- **比例是必填项** 且不能为空或为零
- 使用公式：O = C × L（产量 = 消耗 × 线速度）
- 为不同时间间隔提供材料成分分析

### 第2部分：测试数据分析
- 输入聚氨酯喷涂工艺的实际测试数据
- 必填字段：线速度、测试时间、聚醚重量、异氰酸酯重量
- 可选：聚醚转速和异氰酸酯转速以获得更准确的计算
- 基于实际数据计算比例、产量和消耗

### 第3部分：参数调整
- 使用第2部分的结果计算新参数
- 输入所需的新线速度、消耗和比例
- 提供调整后的组件重量和转速设置
- 显示基于时间的分解以进行操作规划

### 结果
- 所有计算都会自动复制到您的剪贴板
- 结果包括详细分解和材料分析
- 随时切换语言 - 所有结果都将使用您选择的语言

## 获得最佳结果的提示
- 始终确保第1部分中的比例字段具有有效的正数
- 使用第2部分从实际测试数据建立基准参数
- 与第2部分结果一起使用时，第3部分最准确
- 结果经过格式化，便于与您的团队分享

## 技术要求
- 任何现代网络浏览器（Chrome、Firefox、Safari、Edge）
- 无需安装 - 只需打开HTML文件
- 初次加载后可离线工作
